# Fake News Detection using Machine Learning

## Features

- Detect Fake and Real News
- TF-IDF Vectorizer
- Passive Aggressive Classifier
- Flask Web Application
- User Friendly Interface

## Technologies

- Python
- Flask
- Scikit-learn
- HTML
- CSS

## Installation

```bash
pip install -r requirements.txt
```

Run

```bash
python train_model.py
python app.py
```
